### 詳細紀錄

**環境設定流程：**
1. 安裝 Homebrew（/opt/homebrew）→ 加入 ~/.zshrc PATH
2. `brew install gnupg pinentry-mac`
3. `ssh-keygen -t ed25519` → `~/.ssh/id_ed25519_personal`
4. 建立 `~/.ssh/config`（Host github.com-personal alias）
5. `gpg --full-generate-key`（RSA 4096，不過期）→ Key ID: 55B37F93D54FF60A
6. `gpg.program` 設為 `/opt/homebrew/bin/gpg`（解決 git 找不到 GPG 的問題）
7. SSH key + GPG key 上傳 GitHub → commits 顯示 Verified
